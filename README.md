# podkrishey
ПодКрышей — это мессенджер, созданный для тех, кто ценит конфиденциальность, честность и человеческое общение. Здесь нет трекеров, нет сбора данных, нет «пользователей» — только участники, люди и сообщество.

Все сообщения — защищены. Всегда.
Обычные чаты: сквозное шифрование (E2EE) + постквантовая криптография (PQC).
Секретные чаты: напрямую между устройствами — P2P, без серверов, без логов, с дополнительным шифрованием E2EE.

Ключи остаются у вас.
Данные — не продаются.
Переписка — не фильтруется.

Это не «улучшенная версия» существующего.
Это отказ от компромиссов.

В ПодКрышей вы не платите за выдуманную свободу.
Вы получаете настоящую свободу бесплатно.

Безопасный вход
Вход по email-ссылке или через GitHub — без паролей, без утечек. Никакой слежки, никаких cookies, никаких профилей в облаке.

Логотип: крыша + звезда
Символизирует защиту, единство и надежду — то, что объединяет всех, кто под одной крышей.

В ПодКрышей:
Ты не «пользователь» — ты участник.

Ты не «данные» — ты человек.

Ты не «трафик» — ты сообщество.
